--> Overview
A beginner-friendly FastAPI project demonstrating how to build a simple API using Python.
This repository includes endpoints to read patient data and perform diagnosis logic using FastAPI.

--> Features

This FastAPI API includes:

Python-based API built with FastAPI

Loads and returns data from a JSON file

API endpoints to get patient details and run simple diagnosis logic

Auto-generated API documentation via Swagger UI

--> Notes

This project was created for learning purposes to understand FastAPI basics before moving to advanced backend development.
